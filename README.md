# anime1_danmaku

Compatibility:
Firefox/Chrome,
Greasemonkey/Tampermonkey

The sctipt display BiliBili Danmaku in "anime1.me" and "viedeo.eyny.com".

Usage:
1: Copy Danmaku Link from BiliBili site.
2: Paste Danmaku Link to Input in "anime1.me" and "viedeo.eyny.com".
3: press Button.
